The Core Concept of Gartic Phone

Gartic Phone takes the traditional children's game of "Telephone" and elevates it with visual creativity. Here's how it works:

  • Write a Prompt: The first player receives or creates a text prompt that needs to be illustrated.
  • Draw It: The player uses digital drawing tools to create a visual representation of the prompt.
  • Guess It: The next player sees only the drawing and must interpret what it represents, writing a text description.
  • Repeat: This cycle continues through multiple players, creating a chain of transformation.
  • Reveal: At the end, players see how dramatically the original prompt has transformed, often resulting in hilarious misunderstandings.
